Git

Dear Linus,

Thank you so much for this tremendous product you have bestowed upon the world. Your commitment towards sharing one of the greatest collaborative coding tools for free is heroic. Git is a version control system for tracking changes in computer files and coordinating work between multiple users.

Your seamless integration with Github makes for a magical pairing. Using Git from the command line is such a powerful and fun experience. Understanding Git has helped me understand so much more about the workings of the web. Thank you for all you do.

Sincerely,

Aidan

Important Commands

git init - turn a directory into a repository
git status - tells you if any changes have been made in your directory
git rm [filename] - remvoe file
git add . - add all available files to the staging area
git commit -m [message] - commit changes with a message about the edits
git log - view a log of all changes to the repository
git branch - List branches
git checkout - pull the most recent version
git diff - look for differences in files
